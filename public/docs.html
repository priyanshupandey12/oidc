Welcome to the OIDC Provider API. Below is the documentation for all available endpoints to authenticate users and integrate third-party applications.

OpenID Connect Core

GET /.well-known/openid-configuration

OIDC Discovery endpoint. Returns a JSON document containing the provider's configuration, supported scopes, claims, and endpoint URIs.

GET /oauth2/authorize

The authorization endpoint where clients direct the user's browser to authenticate and grant consent.

Query Parameters
ParameterRequiredDescription
client_idYesThe client identifier.
redirect_uriYesWhere to redirect after authorization.
response_typeYesMust be code.
scopeYesSpace-separated list of scopes (e.g. openid profile email).
stateNoOpaque value used to maintain state.
POST /validation/token

Exchanges an authorization code or refresh token for access and ID tokens.

JSON Body Parameters
ParameterRequiredDescription
grant_typeYesauthorization_code or refresh_token
codeIf auth codeThe authorization code received.
refresh_tokenIf refreshThe refresh token.
redirect_uriYesMust match the URI used in authorize.
client_idYesThe client ID.
client_secretYesThe client secret.
GET /info/userinfo

Returns claims about the authenticated user.

Headers
HeaderRequiredDescription
AuthorizationYesBearer <access_token>
GET /public/jwks

Returns the JSON Web Key Set (JWKS) containing public keys used to verify token signatures.

Client Management

POST /clients/register

Registers a new third-party application (OAuth Client).

JSON Body Parameters
ParameterRequiredDescription
nameYesThe name of the application.
redirectUrisYesArray of allowed callback URLs.
allowedScopesYesArray of allowed scopes.

User Authentication

POST /auth/login

Authenticates a user and establishes a session cookie.

JSON Body Parameters
ParameterRequiredDescription
emailYesUser's email.
passwordHashYesUser's password.
POST /auth/register

Registers a new user and logs them in.

JSON Body Parameters
ParameterRequiredDescription
nameYesFull name.
emailYesUser's email.
passwordHashYesUser's password.
